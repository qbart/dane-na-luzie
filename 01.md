## Czego się spodziewać?

- Forma: mini-wprowadzenia -> ćwiczenia indywidualne -> podsumowanie
- Skupiamy się na myśleniu o danych, a nie na suchych definicjach
- Będą proste przykłady
- Nie będziemy wchodzić w szczegóły działania baz danych (chyba, że coś będzie ważne lub chcecie)
- **Nie bójcie się zadawać pytań.**

## Przygotowanie środowiska

### Wariant A (praca na własnym komputerze)

1. Pobrać repozytorium:

https://github.com/qbart/dane-na-luzie

lub skopiować plik na dysk:

https://github.com/qbart/dane-na-luzie/blob/master/docker-compose.yml


2. Uruchomić compose:

```
docker compose up
```

3. Uruchomić w przeglądarce `http://localhost:5430`


4. Zalogować się:

```
db@example.com / secret
```

5. Połączyć się z baza

```
Server name: local
Host name: db
Port: 5432
Database: lms
Password: secret
```

6. Uruchomić query

```sql
SELECT 1
```

### Wariant B (praca w chmurze)

1. dostęp do przeglądarki (https://danenaluzie.qbart.dev)
2. zalogować się swoim username

u1@example.com / secret
...
u15@example.com / secret

3. stworzyć połączenie

```
Server name: local
Host name: db
Port: 5432
User: u1
Database: workshop
Password: secret
```

4. Uruchomić query:

```sql
SELECT 1
```
