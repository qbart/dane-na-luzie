# Dane na luzie

Warsztat będzie poświęcony praktycznemu podejściu do pracy z danymi - od podstawowego modelowania, przez tworzenie zapytań SQL, aż po wykorzystanie AI jako wsparcia analitycznego.
Uczestnicy zobaczą, jak przełożyć pytania biznesowe na strukturę danych i jak krok po kroku dochodzić do użytecznych odpowiedzi. Spotkanie będzie prowadzone w formie krótkich wprowadzeń teoretycznych połączonych z ćwiczeniami indywidualnymi.
Celem jest zbudowanie jasnego, przystępnego sposobu myślenia o danych, który można zastosować w codziennej pracy.
